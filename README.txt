== Description ==
VKontakte Cross-Post – плагин для WordPress, позволяющий публиковать информацию о новых записях
вашего блога на стенах группы или официальной страницы ВКонтакте прямо со страницы публикации статьи. В основе лежит ВКонтакте API,
по средствам которого и производится авторизация и публикация записей на стене вашей группы. Плагин является преемником
плагина Skylark Vkontakte Group Wall Publisher, но претерпел достаточно много изменения. Для того чтобы сохранить
работоспособность предыдущего плагина, я решил осуществить "ребренлинг" и начать поддержку нового плагина.
Для обсуждения и получения дополнительной инфомации посетите наш сайт:<br />
[webstudy.com.ua](http://webstudy.com.ua/vkontakte-cross-post/ "WebStudy.com.ua - совершенствуйся с нами!")

== Installation ==

1. Скачайте архив содержащий файлы плагина с сайта плагина.<br />
2. Загрузите файлы плагина в директорию /wp-content/plugins/.<br />
3. Активируйте плагин в панели администратора.
